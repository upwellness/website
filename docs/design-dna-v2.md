## Top 5 moves to steal

1. **Numeric language as design** (Function) — Every claim quantified. Repeats build cumulative confidence.
2. **Sticky metric anchor** (Whoop) — One signature number visible throughout. For UP Wellness: "Healthspan Years gained" or "ทำให้สุขภาพดีขึ้น X%"
3. **Bento community grid** (Apple) — 12 activities equal treatment. We have 7 → fits perfect.
4. **Named experts section** (Function "Built with top doctors") — Show real coach team with credentials.
5. **Quiz as helpful segmentation** (Blueprint "Find My Protocol") — Already have but reframe softer.

## Restraint principles we should adopt

- Reduce aurora animation intensity (currently too floaty)
- Limit accent color appearances · let cream + ink dominate
- Bigger hero typography (96-120px) · tighter copy
- Body text 16-18px (currently 13-14px) for readability
- More line-height
- Photography placeholder that signals premium (not just colored shapes)

## Specific tactical changes

### Hero
- Remove 3 preview cards (cluttering hero)
- Bigger headline (current 56-72 → 96-120 desktop)
- Move stats below hero, not inside
- Single sticky metric visible (e.g. "−12.3 kg เฉลี่ย")
- 1 primary CTA + 1 ghost · let typography lead

### New sections to add
- **Proof Bar** — numeric anchoring strip · 4-5 metrics with mini-context
- **Manifesto** — "ทำไมเราถึงต่าง" · 3 declarative statements
- **How It Works** — 3-step process (Assess → Plan → Live)
- **Coach Team** — named experts with credentials
- **FAQ** — 6-8 Q&A near footer (trust through candor)

### Sections to upgrade
- **Pillars** — bento-style: 1 big + 2 medium, not 3 equal
- **Community** — bento with featured event card on top
- **Services** — comparison table with "popular" highlight (Whoop tier pattern)
- **Testimonials** — bigger blockquote · transformation story format

### Brand voice in copy
- Per CLAUDE.md: Sage + Everyman + Caregiver
- Per Function: avoid medical jargon in primary headlines
- Per Blueprint: confidence in declarations, not hedged claims
- Per Apple: features explained without overwhelming
